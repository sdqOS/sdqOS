Linux基本命令

ls命令

命令英文原意: list
命令所在路径: /bin/ls
执行权限: 所有用户
功能描述: 显示目录文件

cd命令

命令英文原意: change directory
命令所在路径: shell内置命令
执行权限: 所有用户
功能描述: 切换所在目录
