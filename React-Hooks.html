    Effect Hook

    • Effect Hook 可以让你在函数组中执行副作用操作(用于模拟类组件中的生命钩子) 说白了就是使用生命周期钩

    React中的副作用操作:

    • 发ajax请求数据获取
    • 设置订阅 / 启动定时器
    • 手动更改真实DOM

    语法和说明:

    可以把useEffect Hook 看做这三个函数的组合

    • componentDidMount()
    • componzentDidUpdate()
    • componentWillUnmount()

    Ref Hook

    • 可以在函数组件中储存/查找组件内的标签或任意其它数据
    • 语法: const refContainer = useRef()
    • 作用: 保存标签对象, 功能与React.createRef()一样
