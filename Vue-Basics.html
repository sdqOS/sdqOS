声明式渲染

v-bind:title

v-bind attribute 被称为指令指令带有前缀 v-, 以表示它们是 Vue 提供的特殊 attribute可能你已经猜到了, 它们会在渲染的 DOM 上应用特殊的响应式行为在这里, 该指令的意思是: 将这个元素节点的 title attribute 和当前活跃实例的 message property 保持一致
