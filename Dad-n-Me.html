    神技 小型火焰 (魔法远程)

    • 远程技能: 蓄力释放时一个威力如原子弹般无比强的小型火焰, 适合远程消耗对手, 脸贴脸可使伤害最大化 (伤害高的离谱)

    • 耗蓝量: 0

    • 技能冷却时间: 击中目标后会有1s的冷却时间

    • 学习方法: 击败火拳小子后获得此最强技能

    • 出招表: shift + 移动 + L



    乾坤八卦定 (魔法控制)

    • 控制技能: 释放 一个五行八卦 定住两米以内敌人和自己

    • 耗蓝量: 0

    • 技能冷却时间: 技能使用后会有1s的冷却时间

    • 学习方法: 击败四位摆阵足球运动员后获得此技能

    • 出招表: L



    必学 金盾护体 (霸体免伤)

    • 霸体技能: 给前方敌人一个爱的拥抱 使用技能期间 有一小段位移 期间获得短时间 霸体 和 100%免伤

    • 全部技能霸体: 金盾护体 使用完 然后 疯狂普攻 有几率触发 所有技能都有 霸体 和 100%免伤 受到攻击 “胸前有个 金盾 “ 进入霸体抵消伤害, 前提是出现该效果后, 不再用金盾护体

    • 耗蓝量: 0

    • 技能冷却时间: 0

    • 学习方法: 击败所有修仙篮球队成员后获得此技能

    • 出招表: shift + 移动 + J



    幻直叠影 (增加移速)

    • 强化技能: 放弃上走和下走, 获得幻影特效有几率获得一把剑, 变成四手霸王

    • 耗蓝量: 0

    • 技能冷却时间: 0

    • 学习方法: 击败两名飞车党后获得此技能 后按 L 得到宝剑

    • 出招表: shift + 移动
