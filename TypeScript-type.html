TypeScript学习笔记

TypeScript中的基础静态数据类型有

  • Undefined : 未定义类型;

  • Number : 数值类型;

  • string : 字符串类型;

  • Boolean : 布尔类型;

  • enum : 枚举类型;

  • any : 任意类型, 一个牛X的类型;

  • void : 空类型;

  • Array : 数组类型;

  • Tuple : 元祖类型;

  • Null : 空类型
