    常用complierOptions配置知识

    removeComments 属性

    作用: removeComments的值设置为true, 就是在js中不显示注释

    strict 属性

    作用: 把这个设置为false或者注释掉, 意思是我们可以对设置一些不严格的写法

    noImplicitAny 属性

    作用: 允许你的注解类型 any 不用特意表明

    strictNullChecks 属性

    作用: 不强制检查 NULL 类型

    rootDir属性 和 outDir属性

    作用: 就是存放路径和生成路径

    sourceMap 属性

    作用: 里面储存着位置信息

    noUnusedLocals属性 和 noUnusedParameters属性

    作用: 提示未使用变量和未使用方法
